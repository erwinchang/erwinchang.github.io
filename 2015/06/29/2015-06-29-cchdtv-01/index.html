Return Channel

  • NVR由RS232發出rc command經由同軸線到達camera端
  • Camera端經由RF訊號回傳RC Response
    • 注意當NVR RX收到時，遇到CRC Fail將會drop，此機制(當回傳Response有問題，遇到再送一次command)不確定是否有作

Return Channel Packet Format

  • 實際uart送出訊息如下

  • 檢測到RX板會產生
  • test kit 送出 Device Management Device / Device Info / Get

Cmd為 0x8101
TXID為 0x38ee
RXID為 0xffb0
