蓄電池

  • 蓄電池（Storage battery），俗称电瓶，又稱可充電電池（Rechargeable battery），泛指所有在電量用到一定程度之後可以被再次充電、反覆使用的化學能電池的總稱
  • 可以充電是因為甚化學作用在接上外部電源後其化學作用能反向進行
  • 日常生活中普通使用的有
    • 一次電池相同尺寸（AA/AAA）的鎳氫電池或鎳鎘電池
    • 鋰離子電池
    • 汽車引擎發動時供電的鉛酸電池

鉛酸蓄電池

  • 鉛蓄電池，是蓄電池的一種，電極主要由鉛製成，電解液是硫酸溶液的一種蓄電池
  • 正極板為二氧化鉛板（PbO2），負極板為鉛板（Pb）
  • 鉛酸電池的使用壽命問題
    • 電池是可接受1C的長時間放電[如50Ah 其1C = 50A]
      • 若將電池運用於3C(150A)環境會造成電池壽命快速衰減
    • 啟動型電池需要的是瞬間起動能力，特色是短時間內供應大電流附載使用
      • 電池需具備低內阻，高比重，極板厚度降低等
    • 內阻：越低其大電流放電時不易產生高溫，但缺點是自放電率較高
      • 充飽電後不能置放太久

鎳鎘電池(NiCd)

  • 電壓1.2V，在放電過程電壓也相當平穩
  • 與廣泛應用的碳鋅電池（簡稱碳電）及鹼性電池的1.5V相近
  • 容量高於鉛酸電池，但成本較鉛酸電池高，而且自放電較大
  • 有記憶效應 (就是所謂的第一次充電，一定要充飽)
  • 鎘是有害物質，並不環保
  • 鎳鎘電池wiki

鎳氫電池(NiMH)

  • 是由鎳鎘電池（NiCd battery）改良而來的
  • 鎳氫電池同樣是1.2V，而且容量是鎳鎘電池的兩倍有多，又無記憶效應
  • 環保電池
  • 镍氢电池wiki


圖片來源

鋰離子電池(Li-ion)

  • 鋰離子電池電壓約3.6-3.7V（完全充滿時4.2V）
  • 蓄電池之中能量密度最高者
  • 無記憶效應及低自放電
  • 缺點:
    • 成本高
    • 安全性較差(充電電壓過高會自燃)
    • 充滿電後即使閒置不使用，其容量也會漸漸變小
    • 最差的情況發生在完全充滿時，在室溫25℃的環境中每三個月永久失去的容量為20%
  • 材料
    • 鋰離子電池都是LiCoO2，能量密度較高（也即容量較大）
    • 作鋰離子電池的正極材料主要常見的有：LiCoO2、LiMn2O4、LiNiO2及LiFePO4

電池術語

電池說明

電壓(V)、電池容量(Ah,mAh)與 放電電流(C值)

來源

把電池當水池看…

  • 電壓 = 輸出的水壓有多強 (在馬達能發揮的前提下~電壓越高轉速越快)
    • 馬達端電壓升高，因為內阻固定，因此流過的電流也變大，此時會轉更快，但要小心，不可大於馬達的額定電壓(防止內部電流太大，電感燒掉)
  • 容量 = 水量有多少 (能打多久~相同電量的前提下~電壓或放電數越高~當然能撐的時間就短)
  • C數 = 單位時間內流出的水量 (在馬達能發揮的前提下~電流越大扭力越大)
    • 由馬達的動作流來來決定

你可以這麼想像…電池就是水池~要用於轉動水車輪~就像是用電池驅動馬達~

  • 固定水車輪的大小和轉速的前提下~容量越大的水池可以持續轉動一個水車輪越久~(電量越大~相同配置下電槍打越久)

    • 固定馬達的端電壓下(即固定負載電流)，由電池容量來決定可以跑多久

  • 固定水車輪的大小的前提下~水壓越強可以讓水車輪轉的越快(電壓越高~轉速越快 = 射速越快~但要在馬達的極限範圍內)

    • 固定馬達下，當輸入電壓提高，將會提高轉速 (馬達轉速與端電壓成正比)

  • 固定水車輪的轉速的前提下~水流越大可以轉動越巨大的水車輪~(放電係數越大扭力越大 = 能拉更重的彈簧~但要在馬達的極限範圍內)

    • 固定馬達端電壓，當電池可提供用大的輸出電流時，將可以接更高W的馬達

電池術語

來源

name description
電容量的定義 電池內的電容量
若用電率為1,000mA(毫安培)則該電池可連續使用1小時
這個電量標示其實有其基準，即是以「放電率為0.2C」的條件下所測得的電量 ??
倘若放電率(如0.5C、1.0C 等)更高則會減少可用電量 ??
C表示式 額定容量600mAh的電池，C就是600mA的意思 , 所以電流1C就是600mA
電流2C就是1200mA，電流0.2C 就是120mA
ex. 電池的額定容量為1100mAh時，即表示以1100mA（1C）放電時間可持續1小時
ex. 如以220mA（0.2C）放電時間可持續5小時
能量密度 電池的平均單位體積或質量所釋放出的電能
（Energy density） 鋰離子電池的能量密度是鎳鎘電池的2.5倍，是鎳氫電池的1.8倍
因此在電池容量相等的情況下，鋰離子電池就會比鎳鎘、鎳氫電池的體積更小，重量更輕。
記憶效應 在電池充放電過程中，會在電池極板上產生許多小氣泡，時間一久
這些氣泡會減少電池極板的面積，也間接影響電池的容量
能量轉換效率 充電器對電池充入100分的電力，並不表示電池能放出100分的電力，所以有所謂的能量轉換效率
若充入100分的電力而能放出80分的電力，則轉換效率為80%
鉛蓄電池約為40%~60%
鎳氫電池約為70%
自我放電 電池由於內部會起化學反應的關係，內部會自我放電
（Self discharge） 雖未外接負載但是電池所儲存的電量會隨著時間而逐漸消失
定電壓控制 當偵測到電池電壓上升到設定的電壓準位時就停止充電。這種方式是最簡單的方式
根據電池特性知道電池的電壓上升曲線，就可以將偵測電壓設為最接近充飽時的準位
溫度上升控制 電池容量充到飽和時產生的氧氣會在陰極和鎘反應產生氧化熱，電池的溫度會開始快速上升
利用此現象偵測到溫度到達某程度時既結束充電
容量的估算 電池的容量估算方式很多種，每每總都脫離不了C=IxT的計算方式
電池放電時流過電池的電流大小為I=V/R，V是電池的端電壓，R是負載電阻
電池的放電容量就是為C=V/R*T，T是時間
電池的電壓是隨著放電時間的長短而逐漸降低的
充放電率 所謂充(放)電率是將全部容量的電荷放(充)完所需要的時間，做為充(放)電時的標準速度
二小時率的放電，是指用0.5C 的電流，在二個小時的時間將電池全部容量放完
20分鐘率表示用3C 的電流在20分鐘內將電池額定電量全部放完

  • 如何算是「充電一次」

    • 如果我這次充飽電池後只用了一半的電力，然後又重新把電池充飽，這樣要算幾次呢？
    • 答案是：算0.5次
    • 倘若一個電池可充電200次，以上的動作就允許400次
    • 充飽後只用了10%的電力，然後又再度充飽，如此允許充電200次的電池就可以充2,000次

  • 過充電（Over charge）

    • 在充電過程中電池的電壓會隨著儲存電量的增加而逐漸上升，當電池儲存的電量達到飽和電極材料無法繼續充電時
    • 若繼續充電則電解液會起電解，並且在陽極產生 氧氣，在陰極產生氫氣
    • 如此會在密封的電池內部造成內部壓力上升，會對電池內部結構造成破壞。像這種現象稱之為過度充電

  • 過放電（Over discharge）

    • 電池若是在放電過程中，超過電池放電的終止電壓值
    • 還繼續放電時就可能會造成電池內壓升高，正、負極活性物質的可逆性遭到損壞，使電池的容量產生明顯減少
