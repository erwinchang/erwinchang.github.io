DVB

DVB流程如下

  • 資料來源(video,audio)
  • 資料壓縮(MPEG2)
  • 傳送資料調變(QPS,QAM,COFDM)
  • 傳送路徑(Statellite/Cable/Terrestrial)

圖片來源 : DVB technical overview

DVB System

DVB System

  • 資料來源編碼 (Source Coding)
  • 資料送傳編碼 (Channel Coding)

圖片來源 : 數位電系統原理及軟体技術 pdf

Channel Coding

DVB在傳輸及傳輸方面的編碼

list note
Coding Schemes 1- Error Correction, Emergy Dispersion
(在編碼技術上提供基本的容錯機制) 2- RS/Convolutinoal Code, Interleaving
Transmission Modulation 1- Statellite: QPSK, 8PSK
(DVB不同傳送路徑，編碼方式) 2- Cable: 16/32/64/256-QAM,QPSK
3- Terrestrial: 8VBS(ATSC), COFDM(DVB)

Source Coding

針對節目來源採用的調變(因為要控制TV的播放速度，因此採用MPEG-2壓縮技術)

list note
Source Encoding & Compression Video Compression: MPEG-2 MP@ML(SDTV), MP@HL(HDTV)
Audio Compression: MPEG-1, MPEG-2/AC-3 multi channel, MPEG2-AAC
System Multiplex 1- MPEG2 Transport System protcol
2- MPEG-PSI / DVB-SI

比較表

  • 美規、歐規與日規陸地數位電視廣播系統比較
item note
美規 ATSC 1- 視訊方面採 MPEG-2 技術、音訊則採 Dolby AC-3 標準
2- 調變方式則採用 8 級殘留邊帶(8-Level Vestigial Side-Band ;8-VSB)
3- 傳輸頻寬為 6MHz
歐規 DVB-T 系統 1- 視訊採 MPEG-2 技術、音訊亦是採MPEG-2 技 術
2- 調 變 方 式 採 用 的 是 編 碼 正 交 多 頻 分 工 ( Coded orthogonal Frequency Division Multiplexing;COFDM)
3- 傳輸頻寬為 6MHz / 7MHz / 8MHz
3.1- 所傳送的訊號內有保護頻帶(guard band)設計
3.2- 具有 2k、8k 等模式,配合不同的編碼率(code rate)
日規 ISDB-T 系統 日規 ISDB-T 系統屬於歐規 DVB-T 系統之改良

來源: 陸地數位電視廣播系統調諧器射頻模組之研製

DVB-T

DVB-T Block

  • 由data來源 mpeg-2開始產生，傳送調變編碼
  • 其中加入TPS(確保在播放段可以控速及同步)
  • 名詞定義
    • Forward Error Correction scheme (FEC)
