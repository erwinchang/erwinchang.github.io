slack

  • 先開啟一個討論首頁(xxx.slack.com)
  • 之後可以在此討論此頁面開啟，多個群組討論

功能記錄

  • 指名討論（＠）
    給特定人的訊息(希望他能回應)，使用都可以使用右邊的按鈕(@)來查看是否有人，是需要我回應

  • 在頻道中標記重要討論段落（Pin）
    讓每個人在回顧時可以知道這一個頻道裡最重要的討論在哪裡

  • 星號備忘清單
    星號 功能則是讓各自的成員去收集對自己來說重要的對話

以上這些功能，是比skype好用的地方

如何將bitbucket的commit訊息加到slack

  • 在slack裡面 / 選擇Add an app or integration

  • 在slack裡面 / 先加入bitbucket app，之後再點選Manage查看

  • 在slack裡面 / 查看是否已加入bitbucket app

  • 在slack裡面 / 將下例的Webhook URL即是我們要的url (把它新增到bitbucket裡面的webhook即可)

設定bitbucket commit回應內容

  • 如何設定bitbucket commit其它人的回應內容,hook到slack

    • 選擇Settings/Webhooks

    • 點選之前設定的slack最右邊的Edit

    • 設定Commit comment created
      此時當別人在你的commit有增加說明就可以收到通知
