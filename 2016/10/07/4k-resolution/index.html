名詞定義

畫素(Pixel)

  • 畫素(Pixel)指的就是組成畫面的基本單位，簡單地說，一個畫面是有多少個「點」所組成。比方說一個畫面的水平是800個畫素，而垂直是600個畫素，那麼這個畫面就是800*600=480,000個畫素所組成

影像大小（Image Size）

來源：影像概念

  • 像素（Pixel）
    照片影像是由許多小格子(像素 pixel)組成如下

  • 每個小格子(像素 pixel)決定如下

    • 灰階模式中 1 pixel 相當於 1 byte 的大小
    • RGB 模式中 1 pixel 相當於 3 bytes 的大小
    • CMYK 模式中 1 pixel 相當於 4 bytes 的大小

CMOS active pixel type image sensor with a square pixel array and approximately 2.12M active pixels.
