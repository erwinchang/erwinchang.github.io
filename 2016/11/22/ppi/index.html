ppi算法

PPI全名：「Pixels Per Inch」（pixel：像素，per：每個，inch：英吋），俗稱density也就是每英吋可以有幾個像素

  • ex. ME860的螢幕解析度為960 × 540，螢幕尺寸為4.0英吋，螢幕長寬比為16 : 9，PPI為275。
  • 螢幕尺寸為4.0英吋 (此為對角長),螢幕長寬比為16 : 9
  • 可算出對角長度比例為= sqrt(1616 + 99 ) = 18.357

螢幕長 = 4 x (16/18.357) = 3.486 (英吋)
螢幕寬 = 4 x (9/18.357) = 1.961 (英吋)

縱向PPI = 960 / 3.486 = 275.38
橫向PPI = 540 / 1.961 = 275.36

由此得出我們MOTO ME860的PPI正是等於275

iPhone 4

  • 螢幕尺寸 3.5 吋
  • 螢幕解像度 960 x 640
  • 長寬比為3：2

對角長度比例為= sqrt (33 + 22 ) = 3.60
螢幕長 = 3.5 x (3/3.6) = 2.916(英吋)
螢幕寬 = 3.5 x (2/3.6) = 1.944 (英吋)

縱向PPI = 960 / 2.916 = 329.2
橫向PPI = 640 / 1.944 = 329.2

HTC Butterfly 2

5 吋 1920 x 1080 SLCD
長寬是 14.3cm X 7.1cm

對角長度比例為= sqrt (14.314.3 + 7.17.1 ) = 15.86

螢幕長 = 5 x (14.3/15.86) = 4.5(英吋)
螢幕寬 = 5 x (7.1/15.86) = 2.23(英吋)

縱向PPI = 1920 / 4.5 = 426
橫向PPI = 1080 / 2.23 = 484
