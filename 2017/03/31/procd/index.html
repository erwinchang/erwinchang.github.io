    在OpenWrt規劃下，procd主要取代busybox(initd,klogd,syslogd,watchdog)等功能

  • Init Scripts
    procd的init脚本]

  • openwrt深入学习笔记.pdf
    原生的Linux内核默认启动的第一个用户空间进程是/sbin/init
    Openwrt将其修改为默认启动的第一个用户空间进程为/etc/preinit
