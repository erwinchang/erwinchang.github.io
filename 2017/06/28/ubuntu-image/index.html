  • 使用debootstrap

  • mount boot

  • 增加source.list

  • 設定fstab

  • 產生fstab

  • bind

  • chroot

  • 安裝linux kernel
    apt-cache search linux-image
    apt-cache search linux-headers

  • 安裝grub2

  • 其它
